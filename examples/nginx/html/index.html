Docker Nginx Example

Это пример статической страницы, обслуживаемой Nginx в Docker-контейнере.

Контейнер построен с использованием лучших практик Dockerfile.

Особенности этой конфигурации:

  • Использует nginx:alpine для минимального размера образа
  • Запускается от непривилегированного пользователя
  • Оптимизированы настройки производительности
  • Настроены заголовки безопасности
  • Настроено кэширование статических файлов
  • Добавлен HEALTHCHECK для Docker

Подробности можно найти в GitHub репозитории.
